Sliding Window Pagination

Interactively explore how the sliding window algorithm handles pagination at any scale.

The Algorithm

Always show: first page + last page + a window of pages around the current page. Insert ellipsis between gaps. The window stays compact even with 100+ pages — no scrolling through endless page numbers.

How the Sliding Window Algorithm Works
  1. Always include page 1.
  2. Calculate a window of windowSize pages on each side of the current page.
  3. If the window overlaps with page 1 or the last page, shift it so it stays centered.
  4. Add an ellipsis (…) if there's a gap between page 1 and the window start.
  5. Add an ellipsis if there's a gap between the window end and the last page.
  6. Always include the last page.
